#  Digital Clock

A cute digital clock built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- 🕒 Real-time digital clock
- 📅 Displays the current day and date
- 🌙 AM / PM format
- 💖 Cute My Melody inspired UI
- 🎨 Responsive and clean design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
